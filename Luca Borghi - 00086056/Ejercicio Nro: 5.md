# Ejercicio Nro: 5

## Enunciado

Proponga cinco elementos que ayuden en el desarrollo de software referidos a la propiedad de flexibilidad que se requiere.

Ejemplos:

- Diseño para el cambio.
- Ocultamiento de la información.
- Herramientas de especificación.
- Etc.

## Resolución

Para que un software sea flexible y pueda adaptarse a los cambios durante su desarrollo y mantenimiento, considero importantes los siguientes elementos:

- **Diseño para el cambio:** desarrollar el sistema pensando en que los requerimientos pueden modificarse con el tiempo, facilitando futuras mejoras.
- **Ocultamiento de la información (encapsulamiento):** mantener los detalles internos de cada módulo protegidos para que los cambios no afecten al resto del sistema.
- **Arquitectura modular:** dividir el software en módulos independientes para que cada uno pueda modificarse sin generar grandes impactos en los demás.
- **Documentación clara y actualizada:** contar con documentación que facilite la comprensión del sistema y permita realizar modificaciones de manera más sencilla.
- **Pruebas automatizadas:** realizar pruebas de forma continua para verificar que los cambios no generen nuevos errores y asegurar el correcto funcionamiento del software.
