# Ejercicio Nro: 12

## Enunciado

Utiliza los 7 principios del desarrollo Lean para organizar el cursado y aprobación de este año.

**Qué:**
- Aplica a cada principio a las distintas etapas del proceso de cursado.

**Cómo:**
- Para cada principio indique 3 ejemplos de aplicación que creas conveniente para tu situación como estudiante.
- Agrega un ejemplo más que pueda servir para estudiantes en general.

b) Utiliza un LLM para realizar una tabla de relación de principios.

## Resolución

### Aplicación de los 7 principios del Desarrollo Lean al cursado universitario

### 1. Eliminar los desperdicios (Eliminate Waste)

**Aplicado a mi situación:**
- Organizar un cronograma semanal para evitar perder tiempo decidiendo qué estudiar cada día.
- Resolver primero los trabajos prácticos obligatorios antes de dedicar tiempo a actividades menos importantes.
- Mantener todos los apuntes y archivos organizados en carpetas para encontrarlos rápidamente.

**Ejemplo para cualquier estudiante:**
- Evitar dejar materias o trabajos para último momento, reduciendo el tiempo perdido por desorganización.

### 2. Fomentar la calidad (Build Quality In)

**Aplicado a mi situación:**
- Revisar cada trabajo práctico antes de entregarlo.
- Probar el código varias veces para verificar que funcione correctamente.
- Consultar las devoluciones de los profesores para corregir errores y mejorar en futuras entregas.

**Ejemplo para cualquier estudiante:**
- Leer nuevamente los trabajos antes de entregarlos para detectar errores de contenido o de formato.

### 3. Crear conocimiento (Create Knowledge)

**Aplicado a mi situación:**
- Tomar apuntes durante las clases.
- Elaborar resúmenes antes de cada parcial.
- Practicar ejercicios adicionales para reforzar los conceptos aprendidos.

**Ejemplo para cualquier estudiante:**
- Compartir apuntes y experiencias de estudio con compañeros para aprender de manera colaborativa.

### 4. Decidir lo más tarde posible (Defer Commitment)

**Aplicado a mi situación:**
- Elegir el tema de un proyecto después de analizar todas las opciones disponibles.
- Seleccionar la tecnología a utilizar cuando ya se conocen los requisitos del trabajo.
- Organizar el orden de estudio según las fechas reales de entrega y examen.

**Ejemplo para cualquier estudiante:**
- Esperar a tener toda la información de una consigna antes de comenzar el trabajo.

### 5. Entregar rápido (Deliver Fast)

**Aplicado a mi situación:**
- Completar los trabajos prácticos con anticipación.
- Resolver los ejercicios de cada clase antes de la siguiente.
- Preparar los resúmenes varias semanas antes del examen.

**Ejemplo para cualquier estudiante:**
- Entregar las actividades antes de la fecha límite para disponer de tiempo ante posibles correcciones.

### 6. Respetar a la gente y potenciar el equipo (Respect People, Empower the Team)

**Aplicado a mi situación:**
- Mantener una buena comunicación con mis compañeros de grupo.
- Cumplir con las tareas que me corresponden en los trabajos grupales.
- Ayudar a otros compañeros cuando tengan dificultades con un tema.

**Ejemplo para cualquier estudiante:**
- Escuchar las ideas de todos los integrantes del grupo y distribuir las tareas de forma equitativa.

### 7. Optimizar el conjunto (Optimize the Whole)

**Aplicado a mi situación:**
- Organizar el tiempo considerando todas las materias y no solo una.
- Mantener un equilibrio entre estudio, trabajos prácticos y descanso.
- Priorizar las actividades según su importancia y fecha de entrega.

**Ejemplo para cualquier estudiante:**
- Planificar el cuatrimestre completo para evitar acumulación de trabajos y exámenes.
